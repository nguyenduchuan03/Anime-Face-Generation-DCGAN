# Anime Face Generation using DCGAN
## Project Overview
This project focuses on applying Deep Convolutional Generative Adversarial Networks (DCGAN) to automatically generate realistic anime character face images. It demonstrates practical applications of Deep Learning, Computer Vision, and advanced data preprocessing techniques.

## Core Features & Technologies
- **Frameworks & Libraries:** Python, PyTorch / TensorFlow, NumPy, Matplotlib.
- **Data Preprocessing:** Standardized input images using advanced normalization techniques including Min-Max Scaling and Z-Score Normalization.
- **Model Architecture:** Designed and optimized Convolutional Neural Networks (CNN) for both the Generator and Discriminator, carefully calculating feature map dimensions to ensure stable training.
- **Optimization:** Trained the network using robust optimization algorithms (such as SGD / Adam) to balance the minimax game and improve image generation quality.
